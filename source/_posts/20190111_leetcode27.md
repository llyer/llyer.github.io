问题连接：[27. 移除元素](https://leetcode-cn.com/problems/remove-element/)

```java
public class Solution {
    public int removeElement(int[] nums, int val) {
        //思路，将原来数组a所有不等于val的元素放到b中，返回新数组的长度
        int j=0;
        for(int i=0;i<nums.length;++i){
            if(nums[i]!=val){
                nums[j++]=nums[i];
            }
        }
        return j;
    }
}
```
